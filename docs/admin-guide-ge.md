# ხოზრევანიძის კლინიკის ადმინ-პანელის სახელმძღვანელო

ეს დოკუმენტი აღწერს, როგორ ამართოთ კლინიკის ვებსაიტი Payload CMS-ის
ადმინისტრაციული პანელის გამოყენებით. შენახეთ ეს ფაილი — ის ყველა
ყოველდღიური ამოცანის სრულ ინსტრუქციას შეიცავს.

---

## 1. შესვლა

1. ბრაუზერში გახსენით: **https://khozrevanidze.ge/admin**
   (დემოს ვერსიაზე: **https://clinic-one-blush.vercel.app/admin**)
2. შეიყვანეთ ელფოსტა და პაროლი (გადაცემული თქვენ პირადად)
3. დააწექით „Sign In"-ს

თუ პაროლი დაგავიწყდათ — დარეკეთ ვებსაიტის ადმინისტრატორთან, ის
ხელახლა გენერირებას აკეთებს.

### 1.1. ორფაქტორიანი ავტორიზაცია (2FA / TOTP)

უსაფრთხოებისთვის ადმინ-პანელი მოითხოვს დამატებით 6-ციფრიან კოდს
ყოველი შესვლისას. პირველი შესვლა:

1. ჩამოტვირთეთ ავტენტიფიკატორის აპლიკაცია მობილურზე:
   - **Google Authenticator** (iOS / Android — ყველაზე გავრცელებული)
   - **Microsoft Authenticator** (iOS / Android)
   - **Authy** (iOS / Android / Desktop)
   - **1Password** (თუ უკვე იყენებთ პაროლების მენეჯერად)
2. შესვლის შემდეგ პანელი თქვენ ავტომატურად გადაგამისამართებთ
   „Setup TOTP" გვერდზე, სადაც ნაჩვენებია **QR კოდი**.
3. გახსენით ავტენტიფიკატორის აპლიკაცია → „+" ღილაკი → „Scan QR code"
   → მიუშვით კამერა ეკრანზე გამოსახულ QR-ს.
4. აპლიკაცია გაჩვენებთ 6-ციფრიან კოდს (ის 30 წამში ერთხელ იცვლება).
5. ჩაწერეთ ეს კოდი ვებსაიტზე და დააწექით „Verify"-ს.

ყოველი მომდევნო შესვლა:

1. ჩვეულებრივად შეიყვანეთ ელფოსტა და პაროლი.
2. შემდეგ ეკრანზე გახსენით ავტენტიფიკატორის აპლიკაცია, წაიკითხეთ
   ახლანდელი 6-ციფრიანი კოდი (Khozrevanidze Clinic-ის ქვეშ) და
   შეიყვანეთ.

**მნიშვნელოვანი:**

- არასოდეს გადასცეთ თქვენი QR კოდი ან 6-ციფრიანი კოდი სხვას.
- თუ ტელეფონი დაკარგეთ ან გადააყენეთ — დარეკეთ ვებსაიტის
  ადმინისტრატორთან, ის თქვენი TOTP-ის გადატვირთვას მოახდენს.
- რეკომენდირებულია ავტენტიფიკატორის სარეზერვო ასლის შენახვა
  (Authy და 1Password ამას ავტომატურად აკეთებენ).

## 2. მთავარი პანელი

შესვლის შემდეგ ხედავთ პანელის მთავარ ეკრანს. ის შემდეგი ნაწილებისგან
შედგება:

### 2.1. Doctra სინქრონიზაცია (ვარდისფერი ღილაკი ზემოთ)

**როდის გამოიყენოთ:** როცა Doctra-ში დაამატებთ ახალ ექიმს, ან არსებული
ექიმის სპეციალობა შეიცვალოს.

**როგორ:**
1. დააწექით ღილაკს **„სინქრონიზაცია ახლა"**
2. დაელოდეთ 5–15 წამი (გამოჩნდება სპინერი)
3. შედეგი გამოჩნდება მწვანე/ყვითელ ბარათად:
   - „X ახალი ექიმი დაემატა" — წარმატებული
   - „Y შეცდომა" — ნახეთ შეცდომების სია „იხილე შეცდომები" ღილაკით

**მნიშვნელოვანია:**
- სინქრონიზაცია **ვერ წაშლის** თქვენ მიერ ხელით დამატებულ ფოტოს,
  ბიოგრაფიას ან სხვა სარედაქციო ინფორმაციას
- ის **მხოლოდ ახალ** ექიმებს დაამატებს Doctra-დან
- შეგიძლიათ უსაფრთხოდ გაუშვათ რამდენჯერმე — დუბლიკატებს არ შექმნის

### 2.2. „საჭიროებს ყურადღებას" — 3 ბარათი

ეს ბარათები გიჩვენებენ ექიმებს, რომელთა ინფორმაცია არ არის სრული:

| ბარათი | რას ნიშნავს |
|---|---|
| **🖼 ფოტო აკლია** | ექიმს არ აქვს ფოტო ატვირთული (ნაგულისხმევი ფოტო ჩანს) |
| **✏️ სპეციალობა შესავსებია** | სპეციალობა არ არის შევსებული (ჩანს „—") |
| **🔇 დამალული ექიმები** | ექიმები რომლებიც დროებით დამალულია საიტიდან |

დააწექით ნებისმიერ ბარათს — გადახვალთ უკვე გაფილტრულ ექიმთა სიაში.

### 2.3. სტატისტიკა და სწრაფი მოქმედებები

- **მარცხნივ:** კონტენტის სტატისტიკა (ექიმები, სერვისები, შეფასებები, ა.შ.)
- **მარჯვნივ:** სწრაფი ბმულები — ახალი ექიმის, სიახლის, გვერდის შესაქმნელად

## 3. ექიმების მართვა

გვერდის მისამართი: **/admin/collections/doctors**

### 3.1. ექიმთა სია

ზედა ნაწილში გაქვთ **4 ფილტრის ღილაკი**:
- **ყველა** — სრული სია (146 ექიმი)
- **ფოტო აკლია** — ვისაც ფოტო არ აქვს
- **სპეციალობა შესავსებია** — ვისაც სპეციალობა „—" აქვს
- **დამალული** — ვისი ჩვენებაც გათიშულია

ცხრილში თითო რიგი = ერთი ექიმი. სვეტები:
- **სახელი** — დააწექით ჩასასწორებლად
- **სპეციალობა**
- **განყოფილების ხელმძღვანელი** ✓/✗
- **დამალული** ✓/✗
- **საიტზე ნახვა** (გარე ბმულის ხატულა) — ღია ფანჯარაში ხსნის ექიმის გვერდს

### 3.2. ექიმის რედაქტირება

დააწექით ექიმის სახელზე → იხსნება სარედაქციო გვერდი 3 ჩანართით:

#### იდენტობა (Identity)
- **სახელი** (3 ენაზე — გადართეთ ენა ზედა მარჯვენა კუთხეში)
- **ფოტო** — დააწექით „Add Media" → Upload → აარჩიეთ ფაილი

  > მნიშვნელოვანი: ფოტო ერთხელ იტვირთება და მუშაობს 3-ვე ენაზე.
  > რეკომენდირებული ზომა: 1500×1500 პიქსელი, JPEG, 200KB–1MB.

#### პროფილი (Profile)
- **სპეციალობა** — მთავარი სამედიცინო სპეციალობა
- **ტელეფონი** — ექიმის ნომერი (Doctra-დან მიღებული)
- **ელფოსტა** — Doctra-დან, შეიძლება ცარიელი იყოს
- **ბიოგრაფია** — სრული ტექსტი, შეგიძლიათ გამოიყენოთ მუქი, დახრილი, სათაურები
- **კვალიფიკაცია** — სია (Add Row → მაგ. „კარდიოლოგიის სპეციალისტი")
- **სპეციალიზაცია** — სია (Add Row → მაგ. „დიაგნოსტიკა")
- **გამოცდილების წლები** — რიცხვი
- **ენები** — სია (Add Row → არჩევა dropdown-დან: Georgian, English, Russian, ა.შ.)
- **განყოფილების ხელმძღვანელი** — ჩექბოქსი (ჩაირთვება ჯვარი ვარსკვლავი ექიმის გვერდზე)

#### SEO
- **Meta Title / Description** — საძიებო სისტემებისთვის (Google). თუ ცარიელია,
  ავტომატურად გენერირდება.
- **OG სურათი** — სოციალურ ქსელებში გასაზიარებლად. თუ ცარიელია, ფოტო გამოიყენება.

#### სიდბარი (გვერდის მარჯვენა მხარე)
- **slug** — URL მისამართის ნაწილი. **არ შეცვალოთ** Doctra-დან მიღებულ
  ექიმებზე (URL-ი დაირღვევა).
- **doctraId** — Doctra სისტემის ID. **არ შეცვალოთ.**
- **inactive** — მონიშნეთ, თუ გინდათ ექიმის დროებითი დამალვა
  (წაშლის გარეშე)

### 3.3. სამი ენის რედაქტირება

გვერდის ზედა მარჯვენა კუთხეში არის ენის გადამრთველი:
- **ქართული (ge)** — ნაგულისხმევი
- **English (en)**
- **Русский (ru)**

გადართეთ ენა → ცვლადი ველები (სახელი, სპეციალობა, ბიოგრაფია, ა.შ.)
შეიცვალოს იმ ენაზე. შენახეთ თითო ენაზე ცალ-ცალკე.

> **მნიშვნელოვანი:** ფოტო, slug, ტელეფონი, ელფოსტა, doctraId, inactive,
> გამოცდილების წლები — ეს ველები ერთიანია 3-ვე ენისთვის. სხვა ყველაფერი
> თარგმნადია.

### 3.4. ექიმის ხელით დამატება (Doctra-ში არ არის)

თუ ექიმი ჯერ Doctra-ში არ არის დამატებული, მაგრამ უნდა გამოჩნდეს საიტზე:

1. **/admin/collections/doctors/create**
2. შეავსეთ:
   - სახელი (3 ენაზე)
   - slug (URL-friendly სახელი, მაგ. `lado-baziashvili`)
   - სპეციალობა
   - ფოტო
3. **doctraId-ს არ შეავსოთ** — დატოვეთ ცარიელი
4. დააწექით „Save"

ეს ექიმი გამოჩნდება ექიმთა სიაში, მაგრამ **ჩასაწერი ფორმაში არ ჩაჩნდება**
რადგან Doctra-ში არ არსებობს. როცა Doctra-ში დაამატებენ, წაშალეთ ხელით
შექმნილი რიგი და დააწექით „სინქრონიზაცია"-ს.

## 9. ფოტოების ატვირთვა (Media)

გვერდის მისამართი: **/admin/collections/media**

- ნებისმიერი ფოტო (ექიმის, განყოფილების, სიახლის) აიტვირთება ერთხელ
- მერე გამოიყენება ნებისმიერ ადგილზე (აარჩიეთ ფაილ-არჩევის dropdown-დან)
- ფაილი ინახება **Vercel Blob**-ში (cPanel-ის შემთხვევაში — საიტის სერვერზე)
- **alt** ტექსტი (ფოტოს აღწერა) აუცილებელია 3 ენაზე — Google-სთვის და
  მუშავი მხედველობის სხვა მკითხველებისთვის

> **ფოტოს ზომა:** ფაილს Payload ავტომატურად ცვლის სხვადასხვა ზომებზე
> (thumbnail, card, hero). თქვენ უბრალოდ ატვირთეთ მაღალი ხარისხის ვერსია.

| **ექიმის ფოტოს დამატება** | ექიმთა სია → ექიმის სახელი → იდენტობა → Photo → Upload |
