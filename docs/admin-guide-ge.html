1.1. ორფაქტორიანი ავტორიზაცია (2FA / TOTP)

უსაფრთხოებისთვის ადმინ-პანელი მოითხოვს დამატებით 6-ციფრიან კოდს ყოველი შესვლისას.

პირველი შესვლა

  1. ჩამოტვირთეთ ავტენტიფიკატორის აპლიკაცია მობილურზე:
    • Google Authenticator (iOS / Android — ყველაზე გავრცელებული)
    • Microsoft Authenticator (iOS / Android)
    • Authy (iOS / Android / Desktop)
    • 1Password (თუ უკვე იყენებთ პაროლების მენეჯერად)
  2. შესვლის შემდეგ პანელი თქვენ ავტომატურად გადაგამისამართებთ „Setup TOTP" გვერდზე, სადაც ნაჩვენებია QR კოდი.
  3. გახსენით ავტენტიფიკატორის აპლიკაცია → „+" ღილაკი → „Scan QR code" → მიუშვით კამერა ეკრანზე გამოსახულ QR-ს.
  4. აპლიკაცია გაჩვენებთ 6-ციფრიან კოდს (ის 30 წამში ერთხელ იცვლება).
  5. ჩაწერეთ ეს კოდი ვებსაიტზე და დააწექით „Verify"-ს.
[Screenshot: Setup TOTP page with QR code]
QR კოდი + Enter code ველი + Verify ღილაკი

ყოველი მომდევნო შესვლა

  1. ჩვეულებრივად შეიყვანეთ ელფოსტა და პაროლი.
  2. შემდეგ ეკრანზე გახსენით ავტენტიფიკატორის აპლიკაცია, წაიკითხეთ ახლანდელი 6-ციფრიანი კოდი (Khozrevanidze Clinic-ის ქვეშ) და შეიყვანეთ.
⚠ მნიშვნელოვანი
  • არასოდეს გადასცეთ თქვენი QR კოდი ან 6-ციფრიანი კოდი სხვას.
  • თუ ტელეფონი დაკარგეთ ან გადააყენეთ — დარეკეთ ვებსაიტის ადმინისტრატორთან, ის თქვენი TOTP-ის გადატვირთვას მოახდენს.
  • რეკომენდირებულია ავტენტიფიკატორის სარეზერვო ასლის შენახვა (Authy და 1Password ამას ავტომატურად აკეთებენ).

2.1. Doctra სინქრონიზაცია (ვარდისფერი ღილაკი ზემოთ)

როდის გამოიყენოთ: როცა Doctra-ში დაამატებთ ახალ ექიმს, ან არსებული ექიმის სპეციალობა შეიცვალოს.

როგორ

  1. დააწექით ღილაკს „სინქრონიზაცია ახლა"
  2. დაელოდეთ 5–15 წამი (გამოჩნდება სპინერი)
  3. შედეგი გამოჩნდება მწვანე/ყვითელ ბარათად:
    • „X ახალი ექიმი დაემატა" — წარმატებული
    • „Y შეცდომა" — ნახეთ შეცდომების სია „იხილე შეცდომები" ღილაკით
[Screenshot: Doctra Sync card mid-run]
Spinner + step text: „Fetching departments..." → „Fetching doctors..."
მნიშვნელოვანია
  • სინქრონიზაცია ვერ წაშლის თქვენ მიერ ხელით დამატებულ ფოტოს, ბიოგრაფიას ან სხვა სარედაქციო ინფორმაციას
  • ის მხოლოდ ახალ ექიმებს დაამატებს Doctra-დან
  • შეგიძლიათ უსაფრთხოდ გაუშვათ რამდენჯერმე — დუბლიკატებს არ შექმნის

3.2. ექიმის რედაქტირება

დააწექით ექიმის სახელზე → იხსნება სარედაქციო გვერდი 3 ჩანართით:

იდენტობა (Identity)

  • სახელი (3 ენაზე — გადართეთ ენა ზედა მარჯვენა კუთხეში)
  • ფოტო — დააწექით „Add Media" → Upload → აარჩიეთ ფაილი
ფოტოს რეკომენდაცია ერთხელ იტვირთება და მუშაობს 3-ვე ენაზე. რეკომენდირებული ზომა: 1500×1500 პიქსელი, JPEG, 200KB–1MB.

პროფილი (Profile)

  • სპეციალობა — მთავარი სამედიცინო სპეციალობა
  • ტელეფონი — ექიმის ნომერი (Doctra-დან მიღებული)
  • ელფოსტა — Doctra-დან, შეიძლება ცარიელი იყოს
  • ბიოგრაფია — სრული ტექსტი, შეგიძლიათ გამოიყენოთ მუქი, დახრილი, სათაურები
  • კვალიფიკაცია — სია (Add Row → მაგ. „კარდიოლოგიის სპეციალისტი")
  • სპეციალიზაცია — სია (Add Row → მაგ. „დიაგნოსტიკა")
  • გამოცდილების წლები — რიცხვი
  • ენები — სია (Add Row → არჩევა dropdown-დან: Georgian, English, Russian, ა.შ.)
  • განყოფილების ხელმძღვანელი — ჩექბოქსი (ჩაირთვება ჯვარი ვარსკვლავი ექიმის გვერდზე)
[Screenshot: Doctor edit form — Profile tab]
ფორმის ველები + ლოკალის გადამრთველი ზევით · მარჯვენა სვეტი: slug, doctraId, inactive

SEO

  • Meta Title / Description — საძიებო სისტემებისთვის (Google). თუ ცარიელია, ავტომატურად გენერირდება.
  • OG სურათი — სოციალურ ქსელებში გასაზიარებლად. თუ ცარიელია, ფოტო გამოიყენება.

სიდბარი (გვერდის მარჯვენა მხარე)

  • slug — URL მისამართის ნაწილი. არ შეცვალოთ Doctra-დან მიღებულ ექიმებზე (URL-ი დაირღვევა).
  • doctraId — Doctra სისტემის ID. არ შეცვალოთ.
  • inactive — მონიშნეთ, თუ გინდათ ექიმის დროებითი დამალვა (წაშლის გარეშე)

3.3. სამი ენის რედაქტირება

გვერდის ზედა მარჯვენა კუთხეში არის ენის გადამრთველი:

  • ქართული (ge) — ნაგულისხმევი
  • English (en)
  • Русский (ru)

გადართეთ ენა → ცვლადი ველები (სახელი, სპეციალობა, ბიოგრაფია, ა.შ.) შეიცვალოს იმ ენაზე. შენახეთ თითო ენაზე ცალ-ცალკე.

მნიშვნელოვანი ფოტო, slug, ტელეფონი, ელფოსტა, doctraId, inactive, გამოცდილების წლები — ეს ველები ერთიანია 3-ვე ენისთვის. სხვა ყველაფერი თარგმნადია.

3.4. ექიმის ხელით დამატება (Doctra-ში არ არის)

თუ ექიმი ჯერ Doctra-ში არ არის დამატებული, მაგრამ უნდა გამოჩნდეს საიტზე:

  1. /admin/collections/doctors/create
  2. შეავსეთ:
    • სახელი (3 ენაზე)
    • slug (URL-friendly სახელი, მაგ. lado-baziashvili)
    • სპეციალობა
    • ფოტო
  3. doctraId-ს არ შეავსოთ — დატოვეთ ცარიელი
  4. დააწექით „Save"

ეს ექიმი გამოჩნდება ექიმთა სიაში, მაგრამ ჩასაწერი ფორმაში არ ჩაჩნდება რადგან Doctra-ში არ არსებობს. როცა Doctra-ში დაამატებენ, წაშალეთ ხელით შექმნილი რიგი და დააწექით „სინქრონიზაცია"-ს.
